# Recipe-Page
An HTML/CSS recipe page for Shahi Daal — ingredients list and step-by-step cooking instructions.

A detailed recipe page for Shahi Daal (a rich, royal-style lentil dish), including a full ingredients list and step-by-step cooking instructions.

## Features
- Recipe image and description
- Categorized ingredients list (lentils, base, shahi ingredients, spices, garnish)
- Numbered, detailed cooking steps

## Tech
- HTML5 (nested lists for structured content)
- CSS3 (custom styling, warm food-themed palette)

## What I practiced
- Organizing long-form structured content (ingredients + steps) clearly
- Styling nested lists and headings for readability
